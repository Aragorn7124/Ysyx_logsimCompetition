比赛内容
本次比赛有10个问题，1-5题为基础题，6-10题为进阶题。其问题如下：
问题一：
小明在Logisim中使用基础门电路搭建1位全加器，完成布线后仿真发现电路输出结果存在错误。请找出原电路的逻辑错误并修改，修正后重新仿真验证加法运算与进位输出是否正确。
  示例：
  预期输入：A：1 B：1 SIN：0
  预期输出：S：0 cout：1

问题二：
小明学习完原码加减法运算原理后，需要在Logisim上搭建4位原码加法器，可调用4位加法器、4位减法器、多路选择器等组件完成电路设计。电路需单独引出符号位Y3，运算结果为负数时Y3输出为1，正数则为0。完成布线后仿真发现电路输出结果存在错误。请找出原电路的逻辑错误并修改，修正后重新仿真验证4位原码加法器输出是否正确。
  示例：
  预期输入：A3A2A1A0：0110   B3B2B1B0：1001
  预期输出：Y3Y2Y1Y0：0101
